<h1>Адаптивный лендинг "РумТибет"</h>

<h2>Описание</h2>
<p>Адаптивный одностраничный сайт, реализованный с использованием HTML и SCSS по методологии БЭМ. Проект направлен на отработку семантической вёрстки, построения масштабируемой структуры стилей и адаптации интерфейса под различные устройства и разрешения экранов.</p>

<h3>Стек:</h3>
HTML, SCSS, БЭМ
